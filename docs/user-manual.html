What is TaskMate?

TaskMate is a bot that lives inside Slack. When you post a message — or take a photo of handwritten notes — it reads your words, pulls out every action item, and logs each one as a separate task in Airtable. No forms. No copy-paste. Just post normally and the work gets captured automatically.

Where Do Tasks Go?

🔀
Automatic routing — you don't need to think about this
TaskMate automatically sends every task to the right Airtable table based on who it's assigned to. It also records who logged the task (Assigned By), even if that person isn't the one doing it.
Assignee Airtable Table Digest Format Project Matching
Izzy
elizabeth@oakleyhomebuilders.com		 Tech & Innovation Tasks Grouped by Project, then Priority AI-inferred from context (e.g. "bid messages" → "Bid Automated Reminders")
Dan, and everyone else Operational Tasks Grouped by Category, then Priority Explicit mention required (e.g. "Henderson build", "14 Oak St")
Steve No tasks in Airtable Oversight only Steve's messages are management commands, not tasks
💡
Mixed messages work
If Izzy posts "Dan needs to call the framing sub and I need to fix the MargO security integration", TaskMate creates one record in Operational Tasks for Dan and one in Tech & Innovation Tasks for Izzy — in the same operation. The reply will show: ✅ Logged for Dan (Operational): "Call framing sub" | Logged for Izzy (Tech): "Fix MargO security"
03

Logging Tasks

💬 Method 1 — Post a Message
Type your tasks into any channel TaskMate is in. No special format needed. The bot reads your message and logs every action item as a separate Airtable record.

Add TaskMate to a channel:
Type /invite @TaskMate in any channel.
📸 Method 2 — Photo of Notes
Upload a photo of any handwritten notes — site notepad, whiteboard, sticky note. TaskMate reads the handwriting and logs every task it finds.

On phone: tap the 📎 icon and choose photo.
On desktop: drag and drop into Slack.
💌 Method 3 — DM TaskMate
Open a direct message with TaskMate and type your tasks. They go to your correct Airtable table automatically.

Find TaskMate: Look under Apps in your Slack sidebar, or search for it.
Example — Posting in a channel
👤
Dan
Need to call the framing sub about 14 Oak St ASAP. Also order window hardware this week and follow up with county on Henderson permits.
🤖
TaskMate
✅ Logged for Dan (Operational): "Call framing sub", "Order window hardware", "Follow up Henderson permits"

Commands — In Channels

Mention the bot with @TaskMate followed by your request. Natural language works — you don't need to memorize exact phrasing.

📋   View Tasks
Command
@TaskMate what's my list
What happens
Shows your open tasks in your personal format — Izzy gets tasks by project, everyone else by category and priority.
@TaskMate what's Dan working on
Shows another person's open tasks in their format.
@TaskMate show all open tasks
Full team view — all open tasks from both tables, grouped by person.
@TaskMate show completed
Tasks you completed this week.
✏️   Update Tasks
@TaskMate mark "call framing sub" as done
Sets Status to Done and records today's date. You don't need the exact task name — close enough works.
@TaskMate set "order window hardware" to urgent
Changes priority. Options: Urgent, High, Medium, Low.
@TaskMate assign "call framing sub" to Dan
Reassigns the task and sends Dan a DM notification.
@TaskMate add task: review Jake's Plumbing invoice
Creates a task immediately, assigned to you.
📝   Add Notes & Solutions
@TaskMate add note to "call framing sub": ask about Thursday availability
Adds a note to an Operational task (Dan and others). Good for context or follow-up reminders.
@TaskMate add solution to "fix MargO security": rotated API keys and updated token
Adds a solution description to a Tech & Innovation task (Izzy). Documents how the issue was resolved.
🔍   Scan & Help
@TaskMate scan #permits-channel
Scans the last 100 messages in a channel and logs any tasks found. Also works with "scan this channel" or "scan my recent messages".
@TaskMate help
Posts the full command reference in Slack.

Steve is never assigned tasks in Airtable. When Steve posts in a personal task channel like #dan-tasks or #izzy-tasks, TaskMate interprets it as a management command — not task creation. Steve's updates appear silently in the next digest.

Tips for Best Results

🎯
Be specific about who the task is for
If you post "follow up with the framing sub" without naming a person, the bot assigns it to you. Say "Dan needs to follow up with the framing sub" to assign it to Dan.
🚨
Urgency words set the priority
Urgent: ASAP, today, urgent, critical
High: this week, soon, follow up, need to
Default: Medium, if nothing is mentioned
🔗
Projects link automatically
For Izzy's tasks, the AI infers the project from context. For Dan's tasks, mention the job address or project name explicitly — "Henderson build", "14 Oak St".
📎
Each action item becomes its own record
Post a message with five action items and the bot creates five separate Airtable records. Each can be tracked, reassigned, and completed independently.
✏️
You can always edit in Airtable
The bot logs everything but doesn't lock anything. If a task needs a different project, category, or any other field, open Airtable and edit it directly.
👀
Always check the thread reply
After TaskMate processes your message, it replies in-thread confirming exactly what was logged. If something looks wrong, check the reply before assuming it was captured.
09

What TaskMate Can't Do

🚫
Private 1:1 DMs between two people
Slack doesn't allow bots in private 1:1 conversations. Use a group DM that includes TaskMate, or post in a channel.
🚫
Creating new projects
TaskMate can link tasks to existing projects in Airtable but won't create new Project or Tech Project records. If a task doesn't link to a project automatically, set it in Airtable manually.
🚫
Channels it hasn't been invited to
Type /invite @TaskMate in any channel to add it. It won't pick up tasks from channels it isn't a member of.

Troubleshooting

⚠️
"Something went wrong" message
Try posting again. If it keeps failing, log the task directly in Airtable and let Izzy know so she can check the logs.
📭
Not receiving morning DMs
Make sure you've opened a DM with TaskMate at least once (Slack requires you to initiate). Click TaskMate in the Apps sidebar and send it a message.
🔍
Task not linked to a project
For Izzy's tasks, the AI links by context — if the match isn't obvious, name the project explicitly. For Dan's tasks, always mention the job address or project name. You can also set the project directly in Airtable.
👻
Bot not responding in a channel
Type /invite @TaskMate to add it. If it's already there, check that the channel is public or that TaskMate has been properly invited to the private channel.
